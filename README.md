# DTAP-MPPI
DTAP-MPPI (Dynamic Terrain-Aware Payload MPPI) is an augmented control algorithm based on the Model-Predictive Path Integral (MPPI) Framework. This repository utilizes Numba-CUDA for GPU-accelerated programming in a Python environment. 

## Dependencies
This repository requires the following before usage:
* An NVIDIA GPU supported by Numba
* CUDA Toolkit
* uv (Refer to https://docs.astral.sh/uv/getting-started/installation/ for installation)# DTAP-MPPI
